Política de Privacidade

1. Introdução

Esta política de privacidade descreve como o site PixelCandle coleta, processa e armazena dados pessoais dos usuários, oferecendo serviços profissionais de desenvolvimento de jogos móveis com gráficos e velas interativas. Valorizamos sua privacidade e garantimos a proteção das informações de acordo com os padrões aplicáveis.

2. Quais dados coletamos?

2.1 Dados Pessoais

Ao entrar em contato para nossos serviços, podemos coletar informações como nome, e-mail, telefone e outros dados que você fornece voluntariamente para receber suporte especializado e recomendações sobre desenvolvimento de jogos móveis.

2.2 Informações coletadas automaticamente

O site registra automaticamente informações sobre visitas, incluindo endereço IP, tipo de navegador, sistema operacional, data e hora de acesso, bem como páginas visualizadas. Esses dados ajudam a aprimorar a experiência e os recursos relacionados a gráficos e velas nos jogos móveis.

2.3 Cookies e tecnologias de rastreamento

Utilizamos cookies e tecnologias semelhantes para analisar o comportamento do usuário, salvar preferências e personalizar o conteúdo, oferecendo recomendações mais relevantes para o desenvolvimento de jogos móveis.

3. Como usamos os dados coletados?

3.1 Finalidades principais

Os dados pessoais são usados exclusivamente para fornecer suporte, otimizar processos e enviar informações relevantes sobre gráficos e velas em jogos móveis.

3.2 Melhoria da experiência

A análise das informações permite identificar necessidades, aprimorar recursos e aumentar a eficiência das soluções oferecidas para desenvolvimento de jogos móveis com gráficos e velas.

3.3 Uso de cookies

Cookies ajudam a adaptar o site às suas preferências, melhorar a navegação e fornecer acesso mais fácil aos materiais relacionados ao desenvolvimento de jogos móveis. Ao continuar usando o site, você concorda com o uso dessas tecnologias.

4. Proteção de dados

Aplicamos medidas técnicas e organizacionais para proteger seus dados contra acesso não autorizado, perda ou divulgação. A segurança das informações de nossos usuários é prioridade.

5. Compartilhamento de dados com terceiros

5.1 Base legal

Dados pessoais podem ser compartilhados somente quando necessário para cumprir obrigações legais ou proteger os interesses do site PixelCandle e dos usuários.

5.2 Fornecedores e parceiros

Para operar o site e fornecer suporte, podemos utilizar serviços de terceiros, que devem garantir a confidencialidade e usar os dados apenas para fins autorizados.

6. Direitos dos usuários

6.1 Gerenciamento de dados

Você pode acessar, corrigir ou solicitar a exclusão de seus dados, assim como gerenciar as configurações de cookies através do navegador.

6.2 Contato

Para qualquer dúvida sobre o processamento de dados pessoais, entre em contato pelo e-mail: pixel_candle@gmail.com.

7. Alterações na política

A PixelCandle reserva-se o direito de atualizar esta política. Recomendamos consultar este documento regularmente para se manter informado sobre alterações na proteção de dados.

8. Informações de contato

Para quaisquer questões relacionadas à privacidade e proteção de dados, entre em contato via e-mail: pixel_candle@gmail.com.
